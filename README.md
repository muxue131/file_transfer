# 文件传输系统

一个功能强大的文件传输系统，支持图片、文件和视频传输，以及手机电脑互传功能。

## 功能特性

### 📁 文件传输
- 支持拖拽上传文件
- 支持点击上传文件
- 实时显示上传进度条
- 支持多文件同时上传
- 自动识别文件类型并显示对应图标

### 🎬 视频传输
- 视频流播放功能
- 支持视频文件上传和播放
- 视频播放器集成

### 📱 手机传输
- 二维码扫码连接
- 手机电脑互传文件
- 同一局域网内快速传输

### 🎨 界面设计
- 美观的渐变背景
- 流畅的动画效果
- 响应式设计，支持不同屏幕尺寸
- 现代化的卡片式布局

## 技术栈

### 后端
- **语言**: Go

- **文件存储**: 本地文件系统

### 前端
- **HTML5**: 页面结构

## 使用方法

### 1. 启动服务器

```bash
# 进入项目目录
cd file_transfor

# 运行服务器
go run main.go
```

服务器将在 `http://localhost:8080` 启动。

### 2. 电脑上传文件

1. 打开浏览器，访问 `http://localhost:8080`
2. 点击或拖拽文件到上传区域
3. 查看上传进度和状态
4. 上传完成后，文件会显示在文件列表中

### 3. 手机传输文件

1. 确保手机和电脑在同一局域网
2. 打开电脑端的文件传输系统
3. 查看"手机传输"区域的二维码和本地访问地址
4. 用手机扫描二维码，或在手机浏览器中输入本地访问地址
5. 在手机上选择文件上传
6. 文件会自动传输到电脑的 `uploads` 目录

### 4. 查看文件

1. 上传完成后，文件会显示在"文件上传"区域的文件列表中
2. 点击"查看"按钮在模态框中查看图片、视频或音频
3. 点击"下载"按钮下载文件到本地

## 后端API

### `POST /upload`
- **功能**: 上传文件
- **请求**:  multipart/form-data 格式，包含 file 字段
- **响应**: JSON 格式，包含文件URL和名称

### `GET /api/ip`
- **功能**: 获取本地IP地址
- **响应**: JSON 格式，包含本地IP地址

### `GET /api/files`
- **功能**: 获取已上传的文件列表
- **响应**: JSON 格式，包含文件列表

## 部署说明

### 本地部署

1. 确保安装了 Go 环境
2. 克隆项目到本地
3. 运行 `go run main.go` 启动服务器
4. 访问 `http://localhost:8080`

### 生产部署

1. 编译可执行文件：`go build -o file_transfer_server .`
2. 运行编译后的可执行文件
3. 配置防火墙，允许8080端口访问
4. 可以使用 Nginx 等反向代理进行部署

## 常见问题

### 1. 手机无法访问
- 确保手机和电脑在同一局域网
- 检查防火墙是否阻止了8080端口
- 查看本地访问地址是否正确

### 2. 文件上传失败
- 检查文件大小是否超过限制（50MB）
- 检查网络连接是否稳定
- 查看服务器日志获取详细错误信息

### 3. 二维码无法显示
- 检查网络连接是否正常
- 刷新页面重试
- 查看浏览器控制台是否有错误信息
